Description détaillée

template<typename DataType, int RankValue>
class Arcane::NumArrayBase< DataType, RankValue >

Tableaux multi-dimensionnel pour les types numériques sur accélérateur.

Avertissement
API en cours de définition.

On utilise pour l'instant un UniqueArray2 pour conserver les valeurs. La première dimension du UniqueArray2 correspond à extent(0) et la deuxième dimension au dimensions restantes. Par exemple pour un NumArray<Int32,3> ayant comme nombre d'éléments dans chaque dimension (5,9,3), cela correspond à un 'UniqueArray2' dont le nombre d'éléments est (5,9*3).
