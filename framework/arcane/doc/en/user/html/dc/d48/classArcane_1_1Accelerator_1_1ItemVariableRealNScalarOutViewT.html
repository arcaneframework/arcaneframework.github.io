Detailed Description

template<typename ItemType_, typename Accessor_>
class Arcane::Accelerator::ItemVariableRealNScalarOutViewT< ItemType_, Accessor_ >

Write-only view on a scalar variable of type 'RealN' of the mesh.

This class specializes mutable views for 'Real2', 'Real3', 'Real2x2' and 'Real3x3' reals. The specialization ensures that only the entire vector of reals can be modified. For example:

VariableNodeReal3View force_view = ...;
Node node = ...;
// OK:
force_view[node] = Real3(x,y,z);
// Forbidden:
// force_view[node].x = ...;
