◆ m_use_direct_host_storage

bool Arcane::Accelerator::Impl::GenericFilteringBase::m_use_direct_host_storage = true
protected

Indicates which memory is used for the number of filtered values.

If true, it uses m_host_nb_out_storage directly. Otherwise, it uses m_device_nb_out_storage and performs an asynchronous copy after filtering to copy the value into m_host_nb_out_storage.
