Detailed Description

Constant view on geometric elements of type GeomType::Hexaedron8.

It is possible to retrieve a view of this type via:

  • directly from a Hexaedron8ShapeView instance.
  • directly from a Hexaedron8Element instance
  • a Hexaedron8ElementView instance via Hexaedron8ElementView::constView()
  • a GeomShapeView instance via GeomShapeView::toHexaedron8Element()

For more information on usage, refer to arcanedoc_cea_geometric_viewusage
