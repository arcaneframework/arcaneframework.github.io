Detailed Description

Collection of asynchronous execution queues with priority on accelerator.

The size of the collection is only configurable upon creation and there is a maximum size of POOL_MAX_SIZE. If the requested size is greater than this, the actual size of the collection will be POOL_MAX_SIZE. The element access operator returns the (i % poolSize())-th
