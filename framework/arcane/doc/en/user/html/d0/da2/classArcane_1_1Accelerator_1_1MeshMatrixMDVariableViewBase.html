Detailed Description

template<typename ItemType_, typename MatrixAccessor_, typename Extents>
class Arcane::Accelerator::MeshMatrixMDVariableViewBase< ItemType_, MatrixAccessor_, Extents >

Base class for views of multi-dimensional matrix variable over a mesh item.

MatrixAccessor_ has to be NumMatrixDataViewGetter or NumMatrixDataViewGetterSetter
