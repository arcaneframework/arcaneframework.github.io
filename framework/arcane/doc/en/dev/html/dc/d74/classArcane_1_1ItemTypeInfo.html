Detailed Description

Info on a mesh entity type.

An instance of this class describes a mesh entity type, such as a hexahedral cell, a quadrilateral cell.

See also
ItemTypeMng

There should only be one instance per entity type. Creation of a type is done by the derived class ItemTypeInfoBuilder. Types must be created before any mesh creation (i.e. during the initialization of the architecture).
