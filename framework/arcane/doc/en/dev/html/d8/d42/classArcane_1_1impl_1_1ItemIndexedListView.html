Detailed Description

template<int Extent>
class Arcane::impl::ItemIndexedListView< Extent >

Internal view of an array of entities.

This class is only useful for building entity lists used internally by Arcane. The user version of this class is ItemConnectivityView. The main difference between the two classes is that this one only maintains the lists but does not allow, for example, returning a typed entity.
