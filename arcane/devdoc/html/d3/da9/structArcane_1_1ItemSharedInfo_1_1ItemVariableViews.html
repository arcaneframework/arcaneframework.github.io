Description détaillée

Liste des vues sur les variabes associées aux entités.

Ces variables sont toutes indexables par le localId() de l'entité. Elles sont toujours allouées sauf 'm_parent_ids_view' qui n'est alloué que si l'entité est dans un sous-maillage
