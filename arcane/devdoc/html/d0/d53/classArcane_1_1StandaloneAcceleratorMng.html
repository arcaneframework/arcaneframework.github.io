Description détaillée

Implémentation autonome de 'IAcceleratorMng.h'.

Les instances de cette classe sont créées par ArcaneLauncher::createStandaloneAcceleratorMng().

Cette classe utilise une sémantique par référence.

Cette instance permet d'utiliser les fonctionnalités de Arcane gérant les accélérateurs sans être obligé de créér une application Arcane classique.
