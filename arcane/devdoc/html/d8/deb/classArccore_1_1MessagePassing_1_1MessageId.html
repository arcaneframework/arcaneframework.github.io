Description détaillée

MessageId.

Ces informations sont utilisées pour récupérer les informations d'un message suite à un appel à mpProbe(). L'instance retournée peut-être utilisée pour faire une réception via mpiReceive().

Une fois l'appel à mpProbe() effectué, il est possible de récupérer les informations sur la source du message via sourceInfo().

Avec MPI, cette classe encapsule le type MPI_Message.
