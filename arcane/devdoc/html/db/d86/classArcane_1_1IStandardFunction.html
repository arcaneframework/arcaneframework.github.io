Description détaillée

Interface gérant une fonction standard.

Ces fonctions peuvent être associées à une option du jeu de données.

Les fonctions standards ont les prototypes suivants possibles:

  • f(Real,Real) -> Real
  • f(Real,Real3) -> Real
  • f(Real,Real) -> Real3
  • f(Real,Real3) -> Real3
