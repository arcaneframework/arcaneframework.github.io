virtual void Arcane::IMeshBuilder::fillMeshBuildInfo ( MeshBuildInfo &  build_info)
pure virtual

Remplit build_info avec les informations nécessaires pour créer le maillage.

Certaines valeurs peuvent être remplies par l'appelant mais l'instance peut éventuellement les surcharger. En particulier, il est possible de spécifier la fabrique de maillage à utiliser.

Implémenté dans Arcane::VtkLegacyCaseMeshReader::Builder, Arcane::Cartesian3DMeshGenerator, Arcane::Cartesian2DMeshGenerator, Arcane::Sod3DMeshGenerator, Arcane::HoneyComb3DMeshGeneratorService, et Arcane::HoneyComb2DMeshGeneratorService.

Référencé par Arcane::ArcaneCaseMeshService::createMesh().
