◆ m_use_direct_host_storage

bool Arcane::Accelerator::Impl::GenericFilteringBase::m_use_direct_host_storage = true
protected

Indique quelle mémoire est utilisée pour le nombre de valeurs filtrées.

Si vrai utilise directement m_host_nb_out_storage. Sinon, utilise m_device_nb_out_storage et fait une copie asynchrone après le filtrage pour recopier la valeur dans m_host_nb_out_storage.
