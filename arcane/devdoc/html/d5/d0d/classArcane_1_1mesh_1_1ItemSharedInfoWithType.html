Description détaillée

Classe temporaire pour conserver un ItemSharedInfo et un type d'entité.

Cette classe permet de faire la transition entre la version de Arcane où le ItemSharedInfo contient aussi le type de l'entité et les futures versions où cela ne sera pas le cas.
