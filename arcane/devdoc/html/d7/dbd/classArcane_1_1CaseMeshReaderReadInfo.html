Description détaillée

Informations nécessaires pour la lecture d'un fichier de maillage.

isParallelRead() indique s'il est vrai que tous les rangs sur lequel le maillage est défini vont lire le fichier et qu'il faut donc si possible le distribuer. de manière équilibrée sur l'ensemble des rangs.

format() indique le format du fichier. Par défaut il s'agit de l'extension du nom de fichier. Par exemple, si le fichier est 'toto.vtk', alors le format sera 'vtk'.
