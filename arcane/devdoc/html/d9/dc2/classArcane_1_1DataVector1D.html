Description détaillée

template<typename T>
class Arcane::DataVector1D< T >

Vecteur de données 1D d'un type T.

Cette classe s'utilise par référence, chaque instance gardant une réference sur les autres par l'intermédiaire d'une liste doublement chaînée.

Les opérations add() et resize() obligent à remettre à jour toutes les instances qui font références au même tableau. Elle doivent donc être utilisées le moins possible.
