Description détaillée

Classe représentant une partie d'une adresse d'option du jeu de données. À noter qu'en XML, l'index commence à 1 et non à 0.

Un tag spécial nommé ANY_TAG représente n'importe quel tag. Deux index spéciaux sont aussi disponibles :

  • ANY_INDEX : Représente n'importe quel index,
  • GET_INDEX : Représente un index à récupérer (voir la classe ParameterOptionAddr). Ces élements sont utiles pour l'opérateur ==. À noter que ANY_TAG ne peut pas être définit sans ANY_INDEX. Aussi, le tag ne peut pas être vide.
