Convertit la valeur de la variable d'environnement s en le type T.

Si platform::getEnvironmentVariable(s) est nul, return std::nullopt. Sinon, retourne cette valeur convertie en le type T. Si la conversion n'est pas possible, retourne std::nullopt si throw_if_invalid vaut false ou lève une exception s'il vaut true.

Références ARCANE_FATAL, et Arccore::String::null().
