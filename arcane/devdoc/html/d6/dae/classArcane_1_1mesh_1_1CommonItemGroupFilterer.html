Description détaillée

Classe utilitaire des groupes communs à toutes les parties d'un maillage.

Pour utiliser cette classe, il faut ajouter via addGroupToFilter() les groupes qu'on souhaite filtrer. Il faut ensuite appeler la méthode applyFiltering() pour effectuer le filtrage.
