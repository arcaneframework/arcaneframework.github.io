void Arcane::mesh::TiedInterface::setPlanarTolerance ( Real  tol)
virtual

Definit le seuil relative d'acceptation d'un point projete a une face candidate.

La valeur 0 (ou <0) décrit une acceptation inconditonnelle, une valeur positive définit la borme sup relative à la distance euclidienne séparant deux sommets consécutifs.
