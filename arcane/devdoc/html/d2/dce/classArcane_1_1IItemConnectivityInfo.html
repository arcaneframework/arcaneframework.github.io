Description détaillée

Interface des informations sur la connectivité par type d'entité.

Cette inferface permet de connaître pour un type d'entité donné le nombre maximal d'entitée connectés. Cela peut être utilisé par exemple pour dimensionner des variables.

Les instances de cette interface se récupèrent généralement via IItemFamily::localConnectivityInfos() pour les infos locales au sous-domaine ou IItemFamily::globalConnectivityInfos() pour les infos globales sur tous les maillages.
