Description détaillée

Source d'un jeu de données d'un cas.

Il est possible de positionner soit le nom du fichier (setFileName()) ou directement le contenu (setContent()).

Si content() est vide et que fileName() est non nul, le jeu de données sera lu par Arcane au moment du lancement de l'application.
