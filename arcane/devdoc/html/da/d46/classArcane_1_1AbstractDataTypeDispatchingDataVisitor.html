Description détaillée

IDataVisitor dispatchant les opérations suivant le type de la donnée.

Le dispatcher doit hériter de IDataDispatcherT et de l'interface IDispatcherType passée en argument de la classe template.

L'interface IDispatcherType doit définir le type HasStringDispatch comme TrueType ou FalseType suivant si elle supporte le dispatch du type String.

Il est possible de construire directement une instance via le constructeur ou via la fonction statique create().
