Description détaillée

template<typename T>
class Arccore::SmallSpan< T >

Vue d'un tableau d'éléments de type T.

La vue est non modifiable si l'argument template est de type 'const T'. Cette classe permet d'accéder et d'utiliser un tableau d'éléments du type T de la même manière qu'un tableau C standard. Elle est similaire à Span à ceci près que le nombre d'éléments est stocké sur un 'Int32'.
