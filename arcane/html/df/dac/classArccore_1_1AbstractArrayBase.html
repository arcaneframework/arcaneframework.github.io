Classe de base pour les tableaux.

Cette classe gère uniquement les meta-données pour les tableaux comme le nombre d'éléments ou la capacité.

virtual bool Arccore::AbstractArrayBase::_isUseOwnMetaData ( ) const
inlineprotectedvirtual

Indique si m_md fait référence à m_meta_data.

C'est le cas pour les UniqueArray et UniqueArray2 mais pas pour les SharedArray et SharedArray2.

Réimplémentée dans Arccore::SharedArray2< T >, Arccore::SharedArray< T >, Arccore::SharedArray< XmlNode >, Arccore::SharedArray< Integer >, Arccore::SharedArray< DataType >, Arccore::SharedArray< ItemDirectionInfo >, Arccore::SharedArray< Real >, et Arccore::SharedArray< Int32 >.
