Applique un filtre.

Filtre tous les éléments de input pour lesquels flag est différent de 0 et remplit output avec les valeurs filtrées. output doit avoir une taille assez grande pour contenir tous les éléments filtrés.

L'algorithme séquentiel est le suivant:

Int32 index = 0;
for (Int32 i = 0; i < nb_item; ++i) {
if (flag[i] != 0) {
output[index] = input[i];
++index;
}
}
return index;
