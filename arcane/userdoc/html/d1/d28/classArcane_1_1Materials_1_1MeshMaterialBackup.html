Description détaillée

Sauvegarde/restoration des valeurs des matériaux et milieux.

Les instances de cette classe ne peuvent être utilisées que pour une seule sauvegarde et une seule restauration.

Il est possible de spécifier le nom d'un service de compression pour les données avant les sauvegardes via la méthode setCompressorServiceName(). Si cette méthode n'est pas appelée, la valeur par défaut est celle de IMeshMaterialMng::dataCompressorServiceName().
