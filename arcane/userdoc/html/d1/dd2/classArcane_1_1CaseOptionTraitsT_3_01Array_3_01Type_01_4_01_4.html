template<typename Type>
class Arcane::CaseOptionTraitsT< Array< Type > >

Spécialisation pour les options 'Array'.

Cela est nécessaire car on ne peut pas instancier la classe 'Array'. Pour ce type d'options, il est interdit de modifier les valeurs de l'option donc les vues sont forcément constantes.
