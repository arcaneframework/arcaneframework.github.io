601 // le C++11 utilise le mot clé alignas pour spécifier l'alignement.
602 // Cela fonctionne avec GCC 4.9+ et Visual Studio 2015. Cela ne fonctionne
603 // pas avec Visual Studio 2013. Donc pour Visual Studio on utilise dans tous
604 // les cas __declspec qui fonctionne toujours. Sous Linux, __attribute__ fonctionne
605 // aussi toujours donc on utilise cela. A noter que les structures Simd ont besoin

607 // d'équivalent avec MSVC.
