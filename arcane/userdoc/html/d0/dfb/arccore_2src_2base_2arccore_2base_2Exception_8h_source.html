34/*!
35 * \ingroup Core
36 * \brief Classe de base d'une exception.
37 *
38 * Les exceptions sont gérées par le mécanisme <tt>try</tt> et <tt>catch</tt>
39 * du C++. Toutes les exceptions lancées dans le code <strong>doivent</strong>
40 * dériver de cette classe.
41 *
42 * Une exception peut être collective. Cela signifie qu'elle sera lancée
43 * par tous les processeurs. Il est possible dans ce cas de n'afficher qu'une
44 * seule fois le message et éventuellement d'arrêter proprement l'exécution.
45 */
