Arcane

Cette page contient la documentation pour les utilisateurs de la plate-forme Arcane.

La page Nouvelles fonctionnalités présente la liste des nouveautés de chaque version.

La documentation accessible depuis cette page s'adresse aux personnes désirant connaitre les fonctionnalités de la plate-forme Arcane ou souhaitant développer des modules ou services capables de s'exécuter sur la plate-forme Arcane.

Toutes les personnes développant des modules ou services pour Arcane doivent connaitre au minimum les documents Vue d'ensemble de Arcane et Les types fondamentaux. Ensuite, pour avoir le détail des opérations d'une classe, par exemple pour connaitre l'ensemble des méthodes de la classe Arcane::Cell, il faut consulter la documentation en ligne des codes sources de Arcane.

Les documents disponibles sont:

  • Vue d'ensemble de Arcane : présente briévement les principes et les types de base de Arcane.
  • Les types fondamentaux : présente les notions clefs de Arcane (modules, variables, points d'entrée et services).
  • Le didacticiel ARCANE : présente un didacticiel de Arcane.
  • Les règles de codage : présente les règles de codage utilisées dans Arcane. Pour que les modules développés sur la plate-forme aient une certaine homogénéité, il est intéressant de suivre ces règles.
  • [NEW] Lancement d'un calcul : explique l'initilisation et le lancement d'un calcul
  • Configuration des options jeu de données (fichier AXL) : explique comment paramètrer les modules avec des options utilisateurs fournies dans le jeu de données.
  • La configuration du code : présente le fichier de configuration d'un exécutable réalisé avec la plate-forme Arcane. Ce fichier contient entre autre la description des boucles en temps disponibles.
  • Jeu de données : présente la syntaxe du jeu de données.
  • La boucle en temps : décrit la notion de boucle en temps.
  • Prise en compte du parallélisme dans ARCANE : présente la manière dont Arcane prend en charge le parallèlisme par partionnement de domaine.
  • Concurrence et multi-threading : décrite la notion de concurrence et la parallélisation au niveau des boucles
  • Vectorisation : décrit l'utilisation de la vectorisation (SIMD).
  • Utilisation des traces : décrit comment afficher des traces pendant les exécutions et comment paramétrer leur affichage.
  • [NEW] Utilisation des accélérateurs : présente briévement l'utilisation des accélérateurs dans Arcane.
  • Utilisation du C# avec Arcane : décrit comment utiliser et étendre les fonctionnalités de Arcane en C#.
  • Le support des tests unitaires dans Arcane : décrit comment réaliser des tests unitaires pour les modules et services
  • Gestion des matériaux et des milieux.
  • Boucles sur les entités des matériaux et des milieux
  • Gestion des maillages cartésiens
  • Gestion de la géométrie.

Autres informations:

  • Les types de mailles : décrit les types de maille classiques.
  • Variables d'environnement : liste des variables d'environnement permettant de modifier le comportement de Arcane
  • Détection des problèmes mémoire. : détection des problèmes mémoire.
  • Comparaison bit à bit : comparaison bit à bit de deux exécutions
  • Analyse de performances (Profiling) : décrit les mécanismes disponibles pour l'analyse de performance
  • Utilisation des types tableaux : décrit l'utilisation des types tableaux.
  • Equilibrage de charge sur le maillage : décrit l'utilisation du mécanisme d'équilibrage de charge sur le maillage.
  • Gestion des connectivités des entités décrit le nouveau mécanisme (à partir de la version 2.5) de gestion des connectivités des entités.
