Description détaillée

Postionne le nom de l'action en cours d'exécution.

Le nom d'une action peut-être n'importe quoi. Il est juste utilisé pour différencier les différentes partie d'une exécution et connaître le temps de chacune d'elle. Les actions doivent s'imbriquent les unes dans les autres
