Description détaillée

Tag d'un message.

Le type exact du tag dépend de l'implémentation. Pour être le plus générique possible, on utilise le type 'Int32' qui est aussi le type utilisé couramment avec MPI.

Avec l'implémentation MPI, ce type est utilisé pour le tag MPI et les valeurs maximales autorisées dépendent de l'implémentation. La norme MPI indique seulement qu'il faut au moins autoriser 2^30 (32767) valeurs.

En mode échange de message hybride (MPI + mémoire partagée), la valeur maximale du tag peut être plus faible. Pour toutes ces raisons, il est conseillé de ne pas dépasser la valeur 4096.
