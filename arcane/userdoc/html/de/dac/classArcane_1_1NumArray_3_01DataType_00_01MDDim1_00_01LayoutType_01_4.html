Description détaillée

template<class DataType, typename LayoutType>
class Arcane::NumArray< DataType, MDDim1, LayoutType >

Tableau à 1 dimension pour les types numériques.

Les tableaux à une dimension possèdent l'opérateur 'operator[]' pour compatibilité avec les tableaux classiques du C++.
