Description détaillée

Interface d'écriture des données d'une variable.

Lors d'une écriture, l'ordre d'appel est le suivant:

IDataWriter* writer = ...;
writer->beginWrite(vars)
writer->setMetaData()
foreach(var in vars)
writer->write(var,var_data)
writer->endWriter()
