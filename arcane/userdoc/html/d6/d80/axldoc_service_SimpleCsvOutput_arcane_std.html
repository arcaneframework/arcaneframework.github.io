Arcane  3.7.12.0
'SimpleCsvOutput' service

(implements Arcane::ISimpleTableOutput interface).

Service de sortie de tableau au format CSV.

Summary of options

  • tableDir
  • tableName

Detailed list of options

tableDir (string=) 
<tableDir>value</tableDir>

Nom du répertoire.
Exemple : "MonDossier"
En prenant cet exemple, les fichiers CSV seront enregistrés dans le répertoire ./output/csv/MonDossier/
Peut aussi être définis lors de l'appel à writeFile().
Si vide, alors les fichiers CSV seront écrit dans le répertoire ./output/csv/

tableName (string=) 
<tableName>value</tableName>

Nom du tableau (sera écrit dans la première case du tableau et sera aussi le nom du fichier). Ne pas mettre l'extension ".csv".
Exemple : "MonTableau"
Les 'symboles' de nom sont aussi supporté :
  • @proc_id@ : le rang du processus.
  • @num_procs@ : le nombre total de processus.
Lors de l'écriture, ces symboles seront remplacés par se qu'il représente au moment de l'exécution.
Exemple : "MonTableau_P@proc_id@"
Peut aussi être définis lors de l'appel à init().
